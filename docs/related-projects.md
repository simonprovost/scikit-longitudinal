!!! quote "Auto-prognosis"
    -    _Open-source:_ ‚úÖ
    -    _Authors:_ [VanderSchaar Lab](https://www.vanderschaar-lab.com/)
    -    _Github Link:_ [Auto-prognosis](https://github.com/vanderschaarlab/autoprognosis)

The original Auto-Prognosis documentation suggested longitudinal support via SMS-DKL, but longitudinal workflows do not appear to be available in the codebase today.
